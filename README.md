# 桌面蟑螂（Windows）

这是一个无害的、单文件 Win32 小程序：打开后，一只绘制出的蟑螂会从虚拟桌面的左边自然爬到右边；30 秒后关闭，按 `Esc` 可立即退出。

## 安全边界

- 不读取、写入或删除任何用户文件。
- 不联网、不启动其他程序、不更改系统设置。
- 不需要图片素材，也不需要 Python；蟑螂由程序实时绘制。

## 生成 EXE

在 **Windows** 电脑上双击 `build.bat`，即可在同目录得到 `DesktopCockroach.exe`。它使用微软 C++ 编译器，生成的 EXE 不含运行时依赖。

也可以将本项目推送到 GitHub：工作流会在 Windows 云端构建，并把 `DesktopCockroach.exe` 作为可下载构建产物。

> 提示：Windows 可能对从互联网下载、未经签名的 EXE 显示安全提示。这是系统的常规保护提示；请只分享给明确同意接收该恶搞程序的朋友。
